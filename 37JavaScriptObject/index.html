JavaScript Object

let ชื่อวัตถุ = { propertyName: value } เช่น

let user = {
   name: "Artitaya",age:23
};

let product = { name:"mango", price:150 };

การเข้าถึงข้อมูล

objectName.propertyName
objectName["propertyName"] เช่น

    user.name
    user["name"]

ความแตกต่างของ Array and Object

  • Array มี index เป็นตัวเลข ส่วน Object กำหนดเป็นชื่อ
  • Array ใช้ [] ส่วน Object ใช้ {}

    • JavaScript Object (Method)

    let user = {
       name : "Artitaya",
       age : 23,
       getUser : function() {
       return this.name + "" + this.age ;
       }
    };

    การเรียกใช้งาน

    objectName.methodName() ;
    let data = user.getUser() ;
