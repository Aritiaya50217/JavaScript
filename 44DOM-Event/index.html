DOM Event

   คือ เหตุการณ์หรือการกระทำบางอย่างที่เกิดขึ้นกับ element เช่น การคลิกเมาส์ การเคลื่อนย้ายเมาส์ การกดปุ่มคีย์บอร์ด
โดยผู้พัฒนาสามารถใช event ที่เกิดขึ้นเป็นตัวกำหนดให้ตอสนอง หรือ กระทำบางอย่างได้ เช่น การคลิกแล้วแจ้งเตือน เป็นต้น

ชื่อ Event ความหมาย ทำงานร่วมกับแท็ก
onfocus = " " เมื่อ element ได้รับการ focus select , text , textarea
onblur = " " เมื่อ element นั้นสูญเสียการ focus หรือ ถูกย้าย focus ไปยัง element อื่น select , text , textarea
onchange = " " เมื่อผู้ใช้เปลี่ยนแปลงค่าในฟอร์มรับข้อมูล select , text , textarea
onselect = " " เมื่อผู้ใช้เลือกข้อความ (ใช้เมาส์ลาก) ในช่องข้อความ text , textarea
onsubmit = " " เมื่อผู้ใช้คลิกปุ่ม submit form
onMouseover = " " เกิดเมื่อ object นั้นถูกเลื่อน mouse pointer ไปทับ a , div
onMouseout = " " เกิดเมื่อ objact นั้นถูกเลื่อน mouse pointer ที่ทับอยู่นั้นออกไป a , div
onclick = " " เกิดเมื่อ object นั้นถูกคลิก a , button , checkbox ,radio , reset , submit
onload = " " เกิดเมื่อโหลดเอกสารเสร็จ (รีเฟรช) body
onunload = " " เกิดเมื่อยกเลิกการโหลด เช่น คลิกปุ่ม Stop body
