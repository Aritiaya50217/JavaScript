อาร์เรย์ (Array) คืออะไร

  • ชุดของตัวแปรที่อยู่ในรูปลำดับใช้เก็บค่าข้อมูลให้อยู่ในกลุ่มเดียวกัน ข้อมูลภายในอาร์เรย์จะถูกเก็บบนหน่วยความจำในตำแหน่งที่ต่อเนื่องกัน โดยขนาดของอาร์เรย์จะเล็กหรือใหญ่ขึ้นกับจำนวนมิติที่กำหนดขึ้น

การสร้าง Array

  • สร้างโดยใช้คำสั่ง Array()
    • let ชื่ออาร์เรย์ = new Array();
    • let ชื่ออาร์เรย์ = Array(สมาชิกตัวที่ 1,สมาชิกตัวที่2,...);   เช่น
      let myArray = new Array();
      myArray[0] = 2000 ;
      let days = Array("monday","tuesday","wednesday");

    การเข้าถึงสมาชิก

    ชื่ออาร์เรย์[เลขลำดับ]
