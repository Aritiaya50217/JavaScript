ตัวแปรและชนิดข้อมูล

ตัวแปร คือ ชื่อที่นิยามขึ้นมา เพื่อใช้เก็บค่าข้อมูลสำหรับนำไปใช้งานในโปรแกรม โดยข้อมูลอาจจะประกอบด้วยข้อควา ตัวเลข ตัวอักษรหรือผลลัพธ์จากการประมวลผลข้อมูล

รูปแบบการตั้งชื่อ

  • ขึ้นต้นด้วยตัวอักษรในภาษาอังกฤษตามด้วยตัวอักษรหรือตัวเลข
  • ห้ามขึ้นต้นด้วยตัวเลขหรือสัญลักษณ์พิเศษ
  • ขึ้นต้นด้วย $ (dollar sign)และ _(underscore) ได้
  • มีลักษณะเป็น case sensitive คือ ตัวพิมพ์เล็กพิมพ์ใหญ่จะมีความหมายที่แตกต่างกัน
  • ไม่ซ้ำกับคำสงวน (Keyword)

ตัวแปรใน JavaScript เป็นรูปแบบ Dynamic Typing

  • ตัวแปรแบบ Dynamic Typing คือ ชนิดตัวแปรจะเป็นอะไรก็ได้ตามค่าที่ตัวมันเก็บโดยไม่ต้องประกาศชนิดข้อมูล
  • ตัวแปรปบบ Static Typing ต้องประกาศชนิดข้อมูลในตอนเริ่มต้น เช่น int,double,char เพื่อบอกว่าตัวแปรนี้จะเก็บข้อมูลชนิดไหน

การนิยามตัวแปร Var

  • var (เปลี่ยนแปลงค่าในตัวแปรได้)
  • var ชื่อตัวแปร ;
  • var ชื่อตัวแปร = ค่าเริ่มต้น ;
  • var ชื่อตัวแปร = ค่าเริ่มต้น,ชื่อตัวแปร = ค่าเริ่มต้น

    • การนิยามตัวแปร let

  • let (เปลี่ยนแปลงค่าในตัวแปรได้)
  • let ชื่อตัวแปร ;
  • let ชื่อตัวแปร = ค่าเริ่มต้น ;
  • let ชื่อตัวแปร = ค่าเริ่มต้น,ชื่อตัวแปร = ค่าเริ่มต้น

    • การนิยามตัวแปร const

  • const (ค่าคงที่) เปลี่ยนแปลงค่าไม่ได้
  • const ชื่อตัวแปร = ค่าของตัวแปร ;
