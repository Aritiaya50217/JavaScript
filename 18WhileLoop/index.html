แบบทำซ้ำ (Loop)

กลุ่มคำสั่งที่ใช้ในการวนรอบ (loop) โปรแกรมจะทำงานไปเรื่อย ๆ จนว่าเงื่อนไขที่กำหนดไว้จะเป็นเท็จ จึงจะหยุดทำงาน

  • While
  • For
  • Do...While

    • คำสั่งที่เกี่ยวข้องกับ Loop

  • break  ถ้าโปรแกรมพบคำสั่งนี้จะหลุดจากการทำงานในลูปทันที เพื่อไปทำคำสั่งอื่นที่อยู่นอกลูป
  • continue   คำสั่งนี้จะทำให้หยุดทำการทำงานแล้วย้อนกลับไปเริ่มต้นการทำงานที่ลูปใหม่

    • คำสั่ง While

  • While Loop   จะทำงานตามคำสั่งภายใน while ไปเรื่อย ๆ เมื่อเงื่อนไขที่กำหนดเป็นจริง

    • while (เงื่อนไข) {
        คำสั่งที่จะทำซ้ำเมื่อเงื่อนไขเป็นจริง ;
    }
