โครงสร้างควบคุม(Control Structure)

คือ กลุ่มคำสั่งที่ใช้ควบคุมการทำงานของโปรแกรม

  • แบบลำดับ (Sequence)
  • แบบมีเงื่อนไข (Condition)
  • แบบทำซ้ำ (Loop)

แบบมีเงื่อนไข (Condition)

กลุ่มคำสั่งที่ใช้ในการตัดสินใจในการเลือกเงื่อนไขต่าง ๆ ภายในโปรแกรมมาทำงาน

  • if
  • Switch...Case

รูปแบบคำสั่งเงื่อนไขเดียว

  • if statement

    • เป็นคำสั่งที่ใช้กำหนดเงื่อนไขในการตัดสินใจทำงานของโปรแกรม ถ้าเงื่อนไขเป็นจริงจะทำตามคำสั่งต่าง ๆ ที่กำหนดภายใต้เงื่อนไขนั้น ๆ

      if (เงื่อนไข) {
        คำสั่งเมื่อเงื่อนไขเป็นจริง ;
    }
