คำสั่ง Do...While

  • Do...While

    •     โปรแกรมจะทำงานตามคำสั่งอย่างน้อย 1 รอบ เมื่อทำงานเสร็จจะมาตรวจสอบเงื่อนไขที่คำสั่ง while ถ้าเงื่อนไขเป็นจริงจะวนกลับขึ้นไปทำงานตามคำสั่งใหม่อีกรอบ แต่ถ้าเป็นเท็จจะหลุดออกจากลูป

    โครงสร้างคำสั่ง

    do {
       คำสั่งต่าง ๆ เมื่อเงื่อนไขเป็นจริง ;
    } while (เงื่อนไข) ;
